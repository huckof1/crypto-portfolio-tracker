# Crypto Portfolio Tracker

A simple Python tool that:
- Fetches real-time prices from Binance API
- Calculates current portfolio value and profit/loss (P&L)
- Plots a 7-day price chart for BTC (easily expandable)

## Features
- Real-time prices via Binance public API
- P&L calculation based on your buy prices
- 7-day historical chart using pandas + matplotlib
- Clean console output

## Quick start
```bash
pip install -r requirements.txt
python main.py
